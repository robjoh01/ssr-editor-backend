## Getting Started

Create an `.env` file with the following content:

```bash
PORT=8080
NODE_ENV="<test|production>"
DB_USER="<username>"
DB_PASS="<password>"
```

Then run `npm install` and `npm start`

## Testing

Run `npm test` to run all unit tests and see the coverage report.
